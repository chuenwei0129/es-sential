# es-sential 开发笔记

> 这个项目是我的 JavaScript/TypeScript 工具库学习项目，目标是深入了解现代 npm 包开发的完整流程。

## 📚 学习目标

1. **TypeScript 类型系统**: 深入理解泛型、类型推断、条件类型等高级特性
2. **现代构建工具**: tsup - 零配置的多格式输出构建工具
3. **测试框架**: Vitest - 原生 ESM 支持、TypeScript 开箱即用的测试框架
4. **代码规范**: Biome - 一体式替代 ESLint + Prettier 的新选择
5. **版本管理**: Changesets - 专业的版本和 Changelog 管理工具
6. **TDD (测试驱动开发)**: 通过红-绿-重构循环获得高质量代码
7. **Git Hooks**: Husky + lint-staged 保证代码提交质量
8. **CI/CD**: GitHub Actions 自动发布流程

## 🔄 TDD 工作流程

每实现一个函数都遵循：
1. **Red**: 写测试，看失败
2. **Green**: 写最小代码让测试通过
3. **Refactor**: 重构代码，保持测试通过
